# YTDlpSharp

A modern, user-friendly desktop GUI wrapper for yt-dlp, built with C# and .NET.

## Prerequisites

Before using this application, you must have the following installed and available in your system's PATH:

1. **[yt-dlp](https://github.com/yt-dlp/yt-dlp)** - the actual downloader backend.
2. **[FFmpeg](https://ffmpeg.org/)** - for post-processing (merging, converting).

### How to check if they are installed?
Open a command prompt and type:

```cmd
yt-dlp --version
ffmpeg -version
```

## Building from source

1. Clone the repository.
2. Open the solution file (`YTDlpSharp.slnx`) in Visual Studio 2022 or later.
3. Build the solution (Ctrl+Shift+B).
